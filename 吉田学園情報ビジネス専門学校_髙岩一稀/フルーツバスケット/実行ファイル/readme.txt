〇ゲームプレイ方法
	このゲームはキーボード、ゲームパッドどいらでもプレイ可能です。
	ただし、キーボードはシングルモードでのみ操作できます。
	ゲームパッドはDirectInputのみ対応してます。(エレコム製JC-U3613MシリーズJC-U3613MBKのみチェック済み)
	プレイする際、接続されたゲームパッドの数に応じてプレイヤーの数が決まります。
	マルチモードは１人でプレイすることも可能です。
	
〇アピールポイント
	このゲームは演出にこだわって制作しました。
	特に、ゲーム中のフィーバータイムを表現するのに「画面揺れ」を起こしたところが一番のこだわりポイントです。
	
	苦労したところは、マルチモードのプレイヤー同士の当たり判定です。
	プレイヤーがどちらも動いているときに衝突すると、片方がワープしたり、
	どっかに飛んでいくなど思いもよらないバグに悩まされました。
	この問題は、衝突判定の際に「衝突したプレイヤー双方の現在の位置と前回の位置」を使用して
	「プレイヤーが現在の位置で衝突したら、前回の位置に戻す」ことで解決しました。
