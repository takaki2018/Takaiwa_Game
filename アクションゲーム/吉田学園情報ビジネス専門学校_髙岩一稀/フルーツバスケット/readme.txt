	・ゲームプレイ方法
		このゲームはキーボード、ゲームパッドどいらでもプレイ可能です。
		ただし、シングルモードのみキーボードでプレイできます。
		ゲームパッドはDinputのみ対応してます。
		プレイする際、接続されたゲームパッドの数に応じてプレイヤーの数が決まります。
		１人でプレイすることも可能です。
